¡Regístrate!

Para registrarte debes hacer tres fotografías con la cámara web de tu ordenador. Así podremos crear el modelo biométrico de tu cara. Durante el curso utilizaremos este modelo para contrastar tu identidad. Durante la realización de las actividades, el sistema captará mágenes aleatoriamente. Estas imágenes se enviarán al servidor de Smowltech para verificar tu identidad. Para ello, es necesario que autorices la aplicación a utilizar tu cámara web. Los resultados estarán disponibles en 24 horas como máximo. Las imágenes se revisarán en caso de que hubiera alguna incidencia o reclamación.
Las imágenes se envían siempre con un protocolo de seguridad HTTPS y por canales seguros y cifrados. Además, solo se identifican con un código (nunca con datos de carácter personal). Todas las imágenes se eliminarán cuando termine el semestre.
La UOC sólo revisará las imágenes en los casos de incidencia o reclamación garantizando siempre la seguridad y privacidad de las mismas y el cumplimiento del RGPD. Si se detecta algún comportamiento no autorizado, será comunicado a la UOC.
